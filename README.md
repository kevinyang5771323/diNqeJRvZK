# 图书馆管理系统 java827

## 项目概述

图书馆管理系统是一个基于Java Spring Boot和Vue.js的前后端分离的框架。该系统支持图书借阅与归还功能，并提供相应的源代码、文档，适用于毕业论文或类似项目需求。

## 技术栈

### 前端

- Vue.js

### 后端

- Java
- Spring Boot
- MyBatis-Plus

### 数据库

- MySQL

## 功能模块

- 用户管理
  - 普通用户
  - 超级管理员
- 图书借阅
- 图书归还

## 系统角色

- 普通用户：可以进行图书借阅与归还操作。
- 超级管理员：拥有系统管理权限，包括用户管理和图书管理。

## 运行环境

- 前端：兼容现代Web浏览器
- 后端：JDK 1.8 或以上版本，运行在支持Spring Boot的Servlet容器
- 数据库：MySQL 5.7 或以上版本

## 部署步骤

1. 环境准备：确保JDK和MySQL已正确安装
2. 数据库搭建：执行SQL脚本创建数据库和表
3. 后端部署：编译后端代码，部署到支持Spring Boot的Servlet容器
4. 前端部署：编译前端代码，部署到Web服务器

## 目录结构

```
/
├── backend  后端代码目录
├── frontend 前端代码目录
└── doc      项目文档目录
```

## 核心亮点

- 前后端分离设计，支持独立部署与扩展
- 使用Spring Boot和Vue.js的现代技术栈，保证系统性能与开发效率
- 提供详细的源代码和文档，方便学习和二次开发
